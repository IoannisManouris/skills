# Confidence Rubric

Use confidence ratings to avoid hallucinations and overclaiming.

## High confidence

Use when:

- The claim is directly visible in supplied evidence.
- The claim is visible in official/store materials.
- The claim is supported by source files the user owns.
- Multiple reliable sources agree.

Wording:

- Confirmed
- Observed
- Directly shown

## Medium confidence

Use when:

- The claim is strongly suggested by visible behavior.
- A database or secondary source supports it, but official confirmation is missing.
- Multiple weaker sources agree.

Wording:

- Likely
- Appears to
- This suggests

## Low confidence

Use when:

- The claim is plausible but weakly supported.
- Only one weak/community source supports it.
- It is an implementation inference without code.

Wording:

- Possible
- May
- Could
- Cannot confirm

## Unknown

Use when:

- Evidence is missing.
- Sources conflict.
- The game version is unclear.
- The behavior cannot be observed.

Wording:

- Unknown from available evidence
- Cannot be determined from the supplied materials
